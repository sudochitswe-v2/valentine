# **App Name**: Love Dare

## Core Features:

- Valentine Proposal: Displays the question 'Will you be my valentine?'
- Evasive 'No' Button: The 'No' button moves to a different location when the cursor hovers over it.
- Affirmative Response: Clicking 'Yes' triggers the display of a randomly selected cute text.
- Cute Text Display: Generates a random cute text when the user hits the yes button, which will serve as the valentine day affirmation

## Style Guidelines:

- Primary color: Soft pink (#FFC0CB) to evoke feelings of love and tenderness.
- Background color: Pale blush (#FAE6E7), a desaturated version of the primary, for a gentle backdrop.
- Accent color: Light coral (#F08080) for interactive elements to add warmth.
- Font: 'Belleza' sans-serif for headers and short phrases, giving the app personality suitable for this romantic context, paired with 'Alegreya' serif for longer phrases such as randomly selected cute text
- Use heart-shaped icons for visual elements related to confirming or affirming love.
- Simple, clean layout with a central focus on the question and buttons.
- Subtle, playful animations for the 'No' button movement and the appearance of the cute text.
